# activepoint
sistem digital yang memudahkan guru mencatat dan menilai partisipasi siswa selama pembelajaran. Melalui aplikasi ini, guru dapat menilai aktivitas seperti bertanya, menjawab, berdiskusi, dan bekerja sama. Hasil penilaian direkap otomatis sehingga proses penilaian menjadi lebih cepat, terstruktur, dan objektif.
